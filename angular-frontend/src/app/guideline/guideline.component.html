Backend Debugging Guidelines

  1. In spring-boot only need debug in controller class.
  2. Check all the API from controller and fix the error, then Goto the Angular Error.

Frontend Debugging Guidelines

  1. First create an employee.
  2. After create, follow the instruction from page.
  3. If not created, then try to solve it from Create Employee Components
  4. See inspect console and error message to debug
